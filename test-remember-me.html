Implementation Summary

✅ Frontend Changes:

  • Added rememberMe state to GatewayLogin component
  • Connected checkbox to state with proper event handling
  • Updated login and OTP functions to pass remember me parameter
  • Modified auth store to use localStorage vs sessionStorage based on remember me choice

✅ Backend Changes:

  • Updated validation to accept remember_me parameter
  • Modified token creation to set different expiration times:
    • Remember me = true: 30 days
    • Remember me = false: 24 hours
  • Updated both regular login and OTP login endpoints

✅ Storage Logic:

  • Remember me = true: Uses localStorage (persistent across browser sessions)
  • Remember me = false: Uses sessionStorage (cleared when browser closes)
  • Auth initialization checks both storages
  • Logout clears both storages
